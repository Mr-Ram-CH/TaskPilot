# **App Name**: TaskPilot

## Core Features:

- PM Login: Basic PM login functionality.
- Add Task: PMs can add new tasks with title, description, deadline, and assigned user.
- Edit/Delete Task: PMs can modify or delete existing tasks.
- View Tasks: Users can view tasks assigned to them.
- Update Status: Users can update task status (Pending, In Progress, Done).
- Deadline Notification: Automatically notify the PM if a user misses a task deadline. This tool will check deadlines and determine if and when notifications should be sent.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust.
- Background color: Light gray (#ECEFF1) to ensure readability and a clean interface.
- Accent color: Orange (#FF9800) to highlight important actions and notifications.
- Body and headline font: 'PT Sans' for a modern, humanist look.
- Use clear and intuitive icons from a consistent set, like Material Design Icons.
- Prioritize a clean and well-organized layout. Use cards or lists to display tasks.
- Subtle animations on task status updates to provide feedback to the user.
